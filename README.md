#TABLES AND CHAIRS WEBSITE

#OVERVIEW

Tables and chairs  is an e-commerce website that allows users to browse and purchase furniture online. The platform provides a simple and user-friendly interface where customers can view products, learn about services, and interact with the business.

The project is built using HTML ,CSS and Javasript making it lightweight and easy to customize for beginners learning web development.

#FEATURES

Responsive website design

Homepage with featured products

Services section displayed in interactive boxes

Navigation bar for easy browsing

Product listings for furniture items

Contact section

Simple and clean user interface

checkout cart

payment method

##TECHNOLOGIES USED

You can deploy the website using GitHub Actions:

Go to repository Settings

Click Pages

Select branch main

Save
